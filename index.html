About Me

I'm a passionate DevOps and Platform Engineer with hands-on experience in designing and automating infrastructure pipelines that bridge the gap between development and operations. I specialize in building robust CI/CD pipelines, containerizing applications with Docker, orchestrating workloads on Kubernetes, and provisioning cloud infrastructure using Infrastructure as Code tools like Terraform.

Beyond the DevOps stack, I also build web applications using Python (Django, Flask) and have a solid understanding of HTML and CSS. I believe in writing clean, maintainable code and automating everything that can be automated. I'm constantly learning and pushing the boundaries of what modern platform engineering can achieve.
